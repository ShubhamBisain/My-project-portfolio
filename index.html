Levitating Magnet

This project realises a levitating magnet system that keeps a permanent magnet suspended in mid‑air by balancing gravitational and electromagnetic forces with real‑time feedback control. An Atmega88 microcontroller reads a Hall effect sensor and drives an electromagnet using P, PD, and PID controllers to maintain a precise target height with LEDs indicating stability.

Galvanic Skin Response

This project implements a galvanic skin response (GSR) detector that converts tiny changes in skin resistance into clear visual feedback using LED indicators. An LM358 and LM324 configured as comparators evaluate the GSR‑dependent voltage divider and, through NPN transistor drivers, display different arousal levels on a multi‑LED bar
